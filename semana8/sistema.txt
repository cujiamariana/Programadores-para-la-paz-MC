El comando uname -a muestra información del sistema operativo, como el nombre del sistema, la versión del kernel y la arquitectura del equipo.

El comando whoami muestra el nombre del usuario que tiene la sesión iniciada.

Conocer esta información es útil al administrar un servidor porque permite identificar el sistema que se está utilizando, verificar el usuario que ejecuta los comandos y facilitar tareas de configuración, mantenimiento y seguridad.
